# EasyEvent - Gerência de Configuração de Software
Projeto fictício desenvolvido para a disciplina de Gerência de Configuração de Software da UTFPR.
## Descrição
Sistema de gerenciamento de eventos universitários com cadastro de eventos, inscrições, controle de vagas e emissão de certificados.
